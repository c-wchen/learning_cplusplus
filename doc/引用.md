## 注意事项
1. 引用必须初始化
```c
int a = 10;
int &b = a;    // √
int &b;        // ×
```
2. 引用初始化之后不能改变
```c
int a = 10;
int c = 20;
int &b = a;
b = c;         // ×
```
3. 不要将函数栈上内容当作返回值
4. 本质: **指针常量**

## 应用
1. 变量别名
2. 常量化变量
3. 函数形参
4. 函数返回值
4. 左值
```c
int& fun() {
    static int a = 12;
    return a;
}
// fun() = 12
```
