## 方法重载
1. 同一个作用域
2. 函数名相同
3. 参数个数不同 & 参数类型不同 & 参数顺序不同
满足三个条件涉及的概念就是重载

## 注意
1. 常量引用
```c
// const int 和 int参数类型不同
void fun(int &a) {...}
void fun(const int &a) {...}
```
2. 默认参数
```c
// 二义性, 避免操作
void fun(int a) {...}
void fun(int a, int b = 12) {...}
```

## 运算符重载
1. 成员函数运算符重载
2. 全局函数重载
3. 全局重载必须有一个参数时类或枚举

### 运算符重载
* 左移
* 递增
